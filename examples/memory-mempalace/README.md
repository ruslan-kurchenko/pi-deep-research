# Memory Scout — MemPalace example

Retrieves prior decisions, experiments, and preferences from a [MemPalace](https://mempalaceofficial.com) memory system.

**Enable:** set `"mempalaceUrl"` in `.pi/deep-research/config.json` — the scout is added to the roster automatically.
**Build:** `tsc examples/memory-mempalace/index.ts --outDir examples/memory-mempalace/dist` (or use your project's build step).
