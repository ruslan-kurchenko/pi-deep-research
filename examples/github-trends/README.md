# GitHub Trends Scout — example

Finds trending GitHub repositories related to the brief using the `gh` CLI. No API keys required.

**Enable:** add `"examples/github-trends/index.js"` to `config.scouts[]` in `.pi/deep-research/config.json`.
**Build:** `tsc examples/github-trends/index.ts --outDir examples/github-trends/dist` (or use your project's build step).
